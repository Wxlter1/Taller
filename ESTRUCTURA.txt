FLUJO DE DATOS
==============

1. EDGE DETECTOR (edge_detector.py)
   ↓ Lee video o cámara
   ↓ Detecta estacionamientos con YOLO
   ↓ Envía POST /api/parking/update

2. BACKEND (backend/main.py)
   ↓ Almacena estado de estacionamientos
   ↓ Expone GET /api/parking/status

3. FRONTEND (frontend/src/App.jsx)
   ↓ Polling cada 2 segundos
   ↓ Muestra mapa con iconos (🟢/🔴/🟡)


QUICK START
===========

Terminal 1 - Backend:
  start_backend.bat
  ó
  uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000

Terminal 2 - Edge Detector:
  start_edge_detector.bat
  ó
  python edge_detector.py

Terminal 3 - Frontend:
  start_frontend.bat
  ó
  cd frontend && npm run dev

Abrir navegador: http://localhost:5173


ENDPOINTS API
=============

GET  /api/parking/status
  → Obtiene estado de todos los estacionamientos

POST /api/parking/update
  → Recibe actualizaciones de estado (from edge detector)
  Body: {"spots": [{"spot_id": "spot_0", "status": "free", "confidence": 0.95, "timestamp": 1234567890}]}

GET  /api/parking/spot/{spot_id}
  → Obtiene estado de un estacionamiento específico

POST /detect/
  → Detección por imagen (multipart/form-data)


ESTADOS
=======

"free"     → 🟢 Estacionamiento libre
"occupied" → 🔴 Estacionamiento ocupado
"leaving"  → 🟡 Auto saliendo del estacionamiento
